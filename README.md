# 🥐 Bakery Website

A modern and responsive Bakery Website built using HTML, CSS, and JavaScript. This project showcases a bakery's products, services, and contact information with an attractive user interface.

## ✨ Features

- Responsive design for all devices
- Beautiful bakery-themed UI
- Product showcase section
- About Us section
- Contact information section
- Smooth navigation
- Interactive user experience

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
